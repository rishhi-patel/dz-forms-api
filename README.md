### Env Variables

Create a .env file in then root and add the following

```

PORT = 5000
MONGO_URI = your mongodb uri
JWT_SECRET = 'abc123'

```





### you can use these commands to run backend (all of these commands are mentioned in package.json)
### npm install 
### npm run start
